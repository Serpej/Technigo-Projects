# Description

This is an assigment from technigo to make a Todo app using React with Global State management.

## Steps in the buildning process

 1. Made a sketch on paper with what components I would be needing and how to use the Context API.
 2. Created a React app using Vite in Node.
 3. Made a TaskContainer and a fixed set of tasks using Context API.
 4. Added Task component to be able to view tasks.
 5. Made NewTaskform and logic for adding a new task.
 6. Added TaskCount and made the styling.
 7. Added responsive design.

## Viewing it locallu

  Download the repository and add de dependencies:
  "npm install"
  Then run the code
  "npm run dev"
