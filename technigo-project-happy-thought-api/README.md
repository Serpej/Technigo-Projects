#

In this assignment I was tasked with making a backend for an already set up frontend "X-like" posting website. Basically making it able for the user to post new thoughts, like them and see what posts has already been made and when.

##
1. I set up the needed packages and imported them in server.js.
2. I connected MongoDB.
3. I made a model with certain validation properties matching the frontend.
4. I added an environment variable to refresh the database when used.
5. I started adding endpoints.

###
You can run the project locally by installing the dependencies and then running it in dev mode.
`npm install` and `npm run dev`
